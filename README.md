# judy_couture_ecommerce website

Project Overview
Judy Couture is a prototype e-commerce website developed as part of the Project Module. The website demonstrates the functionality of an online retail platform where users can browse products, view product details, add items to a shopping cart and simulate the checkout process.

The purpose of this project is to demonstrate how front-end web technologies can be used to develop an interactive e-commerce system.

Technologies Used
HTML – used to structure the webpages  
CSS – used to design the layout and styling of the website  
JavaScript – used to implement interactive functionality such as the shopping cart and product filtering

Features
- Product catalogue page
- Product detail page
- Add to cart functionality
- Shopping cart management
- Checkout form
- Order confirmation page
- Product search and filtering

How to Run the Project
1. Download the project files
2. Open the folder in a web browser
3. Open the file:

index.html

4. Navigate through the website to browse products and simulate the checkout process.

Project Structure
index.html – Homepage  
products.html – Product catalogue  
product-detail.html – Product details page  
checkout.html – Checkout page  
thank-you.html – Order confirmation page  
style.css – Website styling  
script.js – Main JavaScript functionality  
cart.js – Shopping cart functionality  
checkout.js – Checkout validation
